# Azure Active Directory Lab

Projeto desenvolvido para estudo de infraestrutura Microsoft utilizando Microsoft Azure.

O objetivo é documentar toda a implantação de um ambiente corporativo contendo Active Directory, DNS, Windows Server, PowerShell, Group Policy, File Server e integração com clientes Windows.

## Tecnologias

- Microsoft Azure
- Windows Server 2022
- Active Directory Domain Services
- DNS
- PowerShell
- RDP

## Arquitetura

Azure

↓

Virtual Network

↓

Windows Server

↓

Active Directory

↓

DNS

↓

Clientes Windows

## Etapas concluídas

- Planejamento da infraestrutura
- Criação do Resource Group
- Criação da Virtual Network
- Criação da Máquina Virtual
- Instalação do Windows Server
- Configuração inicial do servidor
- Instalação do Active Directory
- Promoção para Domain Controller
- Criação da Forest
- Criação do Domínio
- Instalação do DNS
